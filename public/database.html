Database structure & property guide

For the grid to work correctly, your Notion database must use the following property names and types.

Property
Type
Explanation
Name
Title
Main label shown for each item in the grid.
Type
Multi-select
Used to categorize content (Post, Story, Reel, Tiktok).
Publish Date
Date
Controls scheduling and chronological order.
Attachment
Files & Media
Images or files displayed directly inside the grid.
Media/Video
Files & Media
Video files or external video links.
Thumbnail
Files & Media
Preview image shown before opening media.
Pin
Checkbox
Pins an item to the top of the grid.
Hide
Checkbox
Hides an item from the grid without deleting it.

Property names are case-sensitive and must match exactly.

If something doesn’t appear in your grid, it’s usually because one of these properties is missing or named differently.
